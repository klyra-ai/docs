## Introduction

The [**WordPress Integration**](https://klyra.ai/page/wordpress-integration) feature in Klyra AI enables you to seamlessly post AI-generated content to your WordPress websites.

You can:

- **Generate posts** with AI Writer or from saved documents.
- **Publish instantly** to connected WordPress domains.
- **Schedule posts** for a specific date and time.

This integration streamlines your content pipeline, reducing manual copy-paste and giving you full control of your publishing strategy.

## WordPress Account Setup

Before connecting WordPress to Klyra AI, configure your WordPress site:

<Steps>
  <Step title="Connect to your WordPress account">
    - Log in to your **WordPress Admin Dashboard**.
  </Step>
  <Step title="Disable 2FA (if enabled)">
    - Turn off two-factor authentication for smoother integration.
  </Step>
  <Step title="Update Permalink settings">
    - Navigate to **Settings ‚Üí Permalinks**.
    - Select **Post Name**.

      <img
        src="/images/permalinks.webp"
        alt="Permalinks Web"
        class="border rounded-lg shadow-md"
      />
  </Step>
  <Step title="Install Authentication Plugin">
    - Go to **Plugins ‚Üí Add New Plugin**.
    - Install **WREST API Authentication for WP ‚Äì JWT Auth and more** by Mini Orange.
    - After installation, ensure the plugin is **Activated**.

      <img
        src="/images/rest-api-plugin.webp"
        alt="Rest Api Plugin Web"
        class="border rounded-lg shadow-md"
      />
  </Step>
  <Step title="Configure JWT Authentication">
    - Go to **miniOrange API Authentication**.
    - Click **JWT Authentication**.

      <img
        src="/images/authentication-method.webp"
        alt="Authentication Method Web"
        class="border rounded-lg shadow-md"
      />
    - Follow the wizard: **Next ‚Üí Finish**.

      <img
        src="/images/jwt-authentication.webp"
        alt="Jwt Authentication Web"
        class="border rounded-lg shadow-md"
      />
  </Step>
</Steps>

### üí° Tips

- **Always generate content first** before clicking Publish to WordPress.
- **Use scheduling** to build a consistent posting pipeline across multiple domains.
- **Multiple domains:** Add more than one WordPress site if your plan allows ‚Äî perfect for agencies managing client websites.
- **Preview before posting:** Ensure formatting and metadata are correct in the Publish page.
- **Centralized management:** Track all posts (scheduled, live, or drafts) directly inside the WordPress integration panel in Klyra AI.
